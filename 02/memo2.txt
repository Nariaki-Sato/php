練習問題

2.1 
	1. HTMLコードとPHPスクリプトを区切るための記号
		<?php ?>: スクリプティングデリミタ, PHPタグ
	2. PHPスクリプトの区切り文字
		セミコロン(;)
	3. PHPファイルのコメント記法
		- <!-- -->: HTMLコメント
		- //, #: 単一行コメント
		- /* */: 複数業コメント

2.2 
	1. 変数名の構文規則
		(1)$DATA ... 正しいが小文字がいい
		(2)$123 ... 英字か"_"
		(3)$X山田 ... 英字か"_"

2.3 
	1. 定数名
		(1)1A ... 最初の文字は英字にするべき
		(2)data ... 大文字にするべき
		(3)A+1 ... 区切り文字は"_"
	2. 実行中のファイル名・行番号を表示するスクリプト

		<!DOCTYPE html>
		<html>
		<head>
		<meta charset="UTF-8" />
		<title>定数</title>
		</head>
		<body>
		実行中のファイル名：<?= __FILE__ ?><br />
		実行中の行番号：<?= __LINE__ ?>
		</body>
		</html>


2.4
	1. データ型
		boolean, int, float, string, array, object, resource, null
	2. シングルクォート vs ダブルクォート
		- ""は変数展開(Parse)する
		- ""はエスケープシーケンスを認識する

2.5 
	1. 分類
		- インデックスが数値 ... 配列(通常配列)
		- インデックスが文字列 ... 連想配列
	2. その時点での最大インデックス+!が自動的に割り振られる
		$data = [1 => 12, 3 => 10, 5 => 5, ]
		$data[] = 9
		$data[6] = 9


理解度チェック
	1. 間違っているポイント3つ
		<%php
			$ia = 1
			print $1a;
		%>

		正解
		<?php
			$x = 1;
			print $x;
		?>

	2.PHPの概要 
	- PHPは"インタプリタ"型言語
	- コンピュータが理解できる言語に変換("コンパイル")が必要ない
	- HTMLのコードの中に"PHPタグ"を埋め込んで記述
	- 大文字・小文字は区別"しない"
	- PHPファイルの拡張子は".php"

	3. 変数の構文規則: 誤り指摘
	(1)$10 ... 最初の文字が数字
	(2)book ... $で始まってない
	(3)$XYZ ... 正しい
	(4)$msg-1 ... "_"(アンダースコア)
	(5)$kore_demo_hensu ... 英語じゃない

	4. 値引率10%を定義 -> 元の値である500円の支払額を求める
		<?php
			const DISCOUNT = 0.9;
			$price = 500;
			$sum = $price * DISCOUNT
			echo "値引き後の価格は{$sum}円です";
		?>

	5. コード作成
	(1)155.36を指数表記 => 整数にキャスト
		<?php
			var_dump((int)'155.36');
		?>
	(2)「りんご」「みかん」「かき」という要素をもつ配列$fruit
		<?php
			$fruit = ["りんご", "みかん", "かき"];
		?>
	(3)「red/トマト」「yellow/カボチャ」「green/レタス」というキーの組み合わせの連想配列$vegetables
		<?php
			$vegetables = [
				'red' => 'トマト',
				'yellow' => 'カボチャ',
				'green' => 'レタス',
			]
		?>
